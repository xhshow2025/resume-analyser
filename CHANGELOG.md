## v1.3.0 (2026-03-19)

### ✨ 新增功能

#### JD 匹配度分析（求职者专用）
- **核心功能**：深度分析简历与职位的匹配度，帮助求职者评估投递成功率
- **输入方式**：支持 JD 文本粘贴、招聘链接（BOSS 直聘/拉勾/公司官网）、JD 截图（OCR 识别）
- **匹配维度**：
  - 硬性条件匹配 - 学历、专业、工作年限、年龄、地点
  - 技能匹配 - 必备技能、加分技能、工具栈重合度
  - 经验匹配 - 行业背景、项目类型、职责范围
  - 软性素质匹配 - 沟通能力、团队协作、领导力要求
- **输出内容**：
  - 综合匹配度评分（0-100%）
  - 分项匹配度详情（表格形式）
  - 优势项（匹配点）
  - 风险项（不匹配点）及应对策略
  - 简历优化建议
  - 面试准备建议（技术问题 + 行为问题）
- **评分等级**：
  - 90%-100%：非常匹配，强烈建议投递
  - 80%-89%：匹配度高，建议投递
  - 70%-79%：基本匹配，可尝试
  - 60%-69%：匹配度一般，需优化简历
  - 60% 以下：匹配度低，不建议投递

## v1.2.0 (2026-03-19)

### ✨ 新增功能

#### 五维能力评估与可视化雷达图
- **新增五维评估模型**：基于简历内容对候选人进行全方位能力评估
- **五个维度**：
  - 专业技能 - 核心硬技能深度、技术栈广度、专业认证
  - 工作经验 - 工作年限、公司背景、项目复杂度、行业经验
  - 学习能力 - 学历背景、持续学习证据、新技术掌握速度
  - 领导力 - 团队管理经验、跨部门协作、影响力
  - 成就影响力 - 项目成果、业务影响、行业认可（奖项/专利/开源）
- **评分标准**：每个维度 1-10 分，基于简历内容客观评估
- **可视化输出**：使用 Python (matplotlib) 生成雷达图，保存为 PNG 文件
- **中文字体支持**：配置 Windows 黑体，确保中文标签正常显示
- **输出文件**：`resume_radar_chart.png`（8x8 英寸，300 DPI）

## v1.1.2 (2026-03-13)

### ✨ 新增功能

#### 自我介绍模板优化（简历投递专用）
- **问题：** 原模板为面试场景设计，不适合简历投递
- **修复：** 新增 5 个简历投递专用版本
- **格式：** 自然段落式表达，禁止使用有序/无序列表
- **开场白：** 统一加"您好，看了 JD，和我非常匹配。"
- **版本：**
  - 版本一：精简版（100-120 字，简历顶部个人总结）
  - 版本二：详细版（200-250 字，招聘平台在线简历）
  - 版本三：超精简版（50-70 字，字数受限场景）
  - 版本四：BOSS 直聘招呼语（80-100 字）
  - 版本五：邮件正文版（150-180 字）

### 🐛 问题修复

#### 公司推荐链接格式优化
- **问题：** markdown 链接格式在某些平台无法正常跳转
- **修复：** 链接直接显示完整 URL，不使用 markdown 格式
- **格式：** 同时提供公司官网链接和招聘平台搜索链接
- **示例：**
  - 公司官网：https://talent.baidu.com/
  - BOSS 直聘：https://www.zhipin.com/beijing/?query=百度+AI 视觉

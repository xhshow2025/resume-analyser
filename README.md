# Resume Analyser - 简历智能分析 Skills

> 一键分析简历，生成自我介绍，匹配推荐公司与岗位

---

## 🎯 功能概述

本 Skills 为求职者和招聘人员提供智能化简历分析与岗位匹配服务。

**核心功能：**
1. **简历智能分析** - 提取技能、经验、项目、教育背景
2. **自我介绍生成** - 根据简历内容撰写个性化自我介绍（5 个简历投递版本）
3. **公司岗位匹配** - 搜索并推荐匹配的公司和岗位
4. **定制化求职信** - 为每个目标公司生成专属求职信

---

## 📋 输入输出

### 输入
- 简历文件（PDF/DOCX/Markdown 格式）
- 求职偏好（可选）：目标城市、行业、职位类型、薪资范围

### 输出
- **简历分析报告** - 技能图谱、经验总结、优势亮点
- **自我介绍** - 5 个简历投递专用版本（段落式，无列表）
- **公司推荐列表** - 10-20 家匹配公司（含官网 + 第三方平台链接）
- **定制化求职信** - 针对每家公司的专属求职信

## 🛠️ 依赖工具

| 工具 | 用途 |
|------|------|
| `pdf` | PDF 简历内容提取 |
| `docx` | Word 简历内容提取 |

## 📊 分析维度

### 简历分析
- **硬技能** - 编程语言、工具、框架、证书
- **软技能** - 沟通能力、领导力、团队协作
- **经验总结** - 工作年限、行业背景、项目经验
- **教育背景** - 学历、专业、学校、GPA
- **亮点提取** - 突出成就、奖项、专利、论文

### 岗位匹配
- **技能匹配度** - 简历技能 vs 岗位要求
- **经验匹配度** - 工作年限 vs 职位要求
- **行业匹配度** - 过往行业 vs 目标行业
- **薪资匹配度** - 期望薪资 vs 岗位预算

## 🚀 使用流程

### Step 1: 上传简历
用户提供简历文件（PDF/DOCX/Markdown）

### Step 2: 简历解析
自动提取简历内容，结构化存储

### Step 3: 智能分析
- 技能图谱生成
- 经验亮点总结
- 竞争力评估

### Step 4: 自我介绍
生成 5 个简历投递专用版本：
- **版本一** - 精简版（100-120 字，简历顶部）
- **版本二** - 详细版（200-250 字，招聘平台）
- **版本三** - 超精简版（50-70 字，字数受限）
- **版本四** - BOSS 直聘招呼语（80-100 字）
- **版本五** - 邮件正文版（150-180 字）

### Step 5: 公司匹配
搜索并推荐 10-20 家匹配公司，包含：
- 公司名称
- 岗位链接（官网 + 第三方平台）
- 匹配度评分
- 推荐理由

### Step 6: 定制求职信
为每家目标公司生成专属求职信

# Resume Analyser - 简历智能分析 Skills

> 一键分析简历，生成自我介绍，匹配推荐公司与岗位

---

## 🎯 功能概述

本 Skills 为求职者和招聘人员提供智能化简历分析与岗位匹配服务。

**核心功能：**
1. **简历智能分析** - 提取技能、经验、项目、教育背景
2. **五维能力评估** - 专业技能、工作经验、学习能力、领导力、成就影响力，生成可视化雷达图
3. **自我介绍生成** - 根据简历内容撰写个性化自我介绍（5 个简历投递版本）
4. **公司岗位匹配** - 搜索并推荐匹配的公司和岗位
5. **定制化求职信** - 为每个目标公司生成专属求职信
6. **JD 匹配度分析** - 深度分析简历与职位的匹配度，提供优化建议和面试准备指导（求职者专用）

---

## 📋 输入输出

### 输入
- 简历文件（PDF/DOCX/Markdown 格式）
- 求职偏好（可选）：目标城市、行业、职位类型、薪资范围

### 输出
- **简历分析报告** - 技能图谱、经验总结、优势亮点
- **五维能力雷达图** - 5 个维度能力评估可视化图（PNG 图片）
- **自我介绍** - 5 个简历投递专用版本（段落式，无列表）
- **公司推荐列表** - 10-20 家匹配公司（含官网 + 第三方平台链接）
- **定制化求职信** - 针对每家公司的专属求职信
- **JD 匹配度分析报告** - 综合匹配度、分项详情、优势/风险项、优化建议、面试准备（如提供 JD）

## 📊 分析维度

### 简历分析
- **硬技能** - 编程语言、工具、框架、证书
- **软技能** - 沟通能力、领导力、团队协作
- **经验总结** - 工作年限、行业背景、项目经验
- **教育背景** - 学历、专业、学校、GPA
- **亮点提取** - 突出成就、奖项、专利、论文

### 五维能力评估
- **专业技能** - 核心硬技能深度、技术栈广度、专业认证（1-10 分）
- **工作经验** - 工作年限、公司背景、项目复杂度、行业经验（1-10 分）
- **学习能力** - 学历背景、持续学习证据、新技术掌握速度（1-10 分）
- **领导力** - 团队管理经验、跨部门协作、影响力（1-10 分）
- **成就影响力** - 项目成果、业务影响、行业认可（1-10 分）

### 岗位匹配
- **技能匹配度** - 简历技能 vs 岗位要求
- **经验匹配度** - 工作年限 vs 职位要求
- **行业匹配度** - 过往行业 vs 目标行业
- **薪资匹配度** - 期望薪资 vs 岗位预算

## 🚀 使用流程

### Step 1: 上传简历
用户提供简历文件（PDF/DOCX/Markdown）

### Step 2: 简历解析
自动提取简历内容，结构化存储

### Step 3: 智能分析
- 技能图谱生成
- 经验亮点总结
- 竞争力评估
- 五维能力评分

### Step 4: 生成五维雷达图
使用 Python (matplotlib) 生成可视化雷达图：
- 计算五个维度得分（1-10 分）
- 绘制雷达图
- 配置中文字体支持
- 保存为 PNG 文件（resume_radar_chart.png）

### Step 5: 自我介绍
生成 5 个简历投递专用版本：
- **版本一** - 精简版（100-120 字，简历顶部）
- **版本二** - 详细版（200-250 字，招聘平台）
- **版本三** - 超精简版（50-70 字，字数受限）
- **版本四** - BOSS 直聘招呼语（80-100 字）
- **版本五** - 邮件正文版（150-180 字）

### Step 6: 公司匹配
搜索并推荐 10-20 家匹配公司，包含：
- 公司名称
- 岗位链接（官网 + 第三方平台）
- 匹配度评分
- 推荐理由

### Step 7: 定制求职信
为每家目标公司生成专属求职信

### Step 8: JD 匹配度分析（求职者专用）
如用户提供职位描述（JD），进行深度匹配分析：
- 解析 JD 内容（硬性条件、技能要求、职责描述、软性素质）
- 计算匹配度（综合评分 + 分项匹配）
- 识别优势项和风险项
- 提供简历优化建议
- 提供面试准备建议

### JD 匹配度分析报告
```markdown
# JD 匹配度分析报告

## 职位信息
- **公司：** XX 科技有限公司
- **职位：** AIGC 视觉设计师
- **地点：** 深圳
- **薪资：** 25-40k/月

## 综合匹配度：88%（建议投递）

## 分项匹配详情
| 维度 | 匹配度 | 说明 |
|------|--------|------|
| 硬性条件 | 95% | 学历硕士（要求本科）、年龄 27 岁（符合） |
| 技能匹配 | 90% | SD/MJ/ControlNet 全部掌握 |
| 经验匹配 | 85% | 项目质量高，年限略短 |
| 软性素质 | 80% | 协作经验足，管理经验待加强 |

## 优势项 ✅
1. AI 工具栈高度匹配
2. 多模态能力突出
3. 学历背景优秀

## 风险项 ⚠️
1. 工作年限略短（应对：强调项目密度）
2. 缺少团队管理经验（应对：强调主导经验）

## 简历优化建议
1. 突出 AIGC 专注度
2. 补充项目数据
3. 增加作品集链接

## 面试准备建议
- 技术问题：ControlNet 角色一致性、LoRA 训练
- 行为问题：项目挑战、效率与质量平衡
```

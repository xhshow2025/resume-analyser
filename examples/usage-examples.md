# Resume Analyser 使用示例

---

## 示例 1: 应届毕业生求职

### 输入
```
用户：帮我分析简历，我是 2026 届应届毕业生，学计算机的

[上传简历 PDF]
```

### 输出
1. **简历分析报告** - 突出学习能力和项目经验
2. **自我介绍** - 强调潜力而非经验
3. **公司推荐** - 校招友好的公司
4. **求职信** - 针对应届生岗位

## 示例 2: 职场跳槽

### 输入
```
用户：我想跳槽，现在在北京做前端，5 年经验，帮我看看机会

[上传简历 PDF]
用户：期望薪资 40-50k，目标公司字节、阿里、腾讯
```

### 输出
1. **简历分析报告** - 突出技术深度和项目成果
2. **自我介绍** - 强调业务价值和技术影响力
3. **公司推荐** - 字节、阿里、腾讯等大厂
4. **求职信** - 针对每家公司定制

## 使用技巧

### 1. 提供详细信息
```
✅ 好：我有 5 年前端经验，现在北京，期望 40-50k
❌ 差：帮我看看机会
```

### 2. 明确目标
```
✅ 好：我想投字节的前端专家岗位
❌ 差：有什么公司推荐吗
```

### 3. 及时反馈
```
✅ 好：面试挂了，面试官说我 XX 方面不足
❌ 差：[无反馈]
```

## 常见问题

### Q: 简历分析准吗？
A: 基于简历内容客观分析，建议结合人工判断。

### Q: 公司推荐是实时的吗？
A: 基于搜索结果的推荐，建议投递前再次确认岗位是否还在招聘。

### Q: 求职信需要每封都定制吗？
A: 强烈建议定制，一封通用求职信成功率很低。

### Q: 多久能找到工作？
A: 因人而异，一般 1-3 个月。市场好时更快。

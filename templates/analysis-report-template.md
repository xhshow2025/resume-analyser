## 求职建议

### 简历优化
1. **突出技术深度** - 增加 React 源码理解、性能优化细节
2. **量化成果** - 所有成就用数据说话
3. **精简描述** - 删除无关内容，聚焦核心优势
4. **格式优化** - 使用简洁专业的模板

3. **分批投递** - 先投几家练手，再投目标公司
4. **定制简历** - 针对每家公司调整重点

1. **市场调研** - 了解目标公司薪资范围
2. **底线设定** - 确定最低接受薪资
3. **期望报价** - 比底线高 15-20%
4. **综合考量** - 薪资 + 股票 + 福利 + 发展

## 使用说明

### 填写规范
1. **所有成就必须量化** - 用数据说话
2. **技能标注年限** - 体现熟练程度
3. **项目突出难点** - 展示解决问题能力
4. **评分客观公正** - 不夸大不贬低

### 更新频率
- 每次面试后更新反馈
- 每周复盘投递效果
- 每月调整求职策略

### 保密要求
- 不在公开场合分享
- 删除时彻底销毁
- 不转发给第三方

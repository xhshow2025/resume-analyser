# 求职信模板库

---

## 标准模板

```markdown
尊敬的 [公司名称] 招聘团队：

您好！

我在 [渠道] 看到了贵公司 [职位名称] 的招聘信息，
经过仔细研究岗位要求和公司介绍，
我认为我的技能和经验与该职位高度匹配，特此投递简历。

【为什么选择 [公司名称]】

我一直关注贵公司在 [领域] 的发展，特别是 [具体产品/技术/成就]。
贵公司 [文化理念/技术追求/业务方向] 与我的职业价值观高度契合。
[可选：使用过贵公司的产品，体验非常好/认同贵公司的技术博客观点]

【为什么选择我】

根据岗位描述，您正在寻找具备以下能力的人才：

1. [岗位要求 1]
   - 我在 XX 公司有 X 年相关经验
   - 主导过 XX 项目，实现 XX 成果

2. [岗位要求 2]
   - 我熟练掌握 XX 技术栈
   - 有 XX 认证/开源贡献/技术博客

3. [岗位要求 3]
   - 我带领过 X 人团队
   - 建立了 XX 机制，提升了 XX 效率

【核心成就】

我最想分享的是 [与岗位最相关的成就]。

当时面临 [挑战/背景]，
我通过 [行动/方案]，
最终实现 [成果 - 用数据说话]。

这个经历让我 [收获/成长]，
也证明了我具备 [岗位需要的能力]。

【期待合作】

我相信我的以下能力能够为贵团队带来价值：
- [核心优势 1] - 具体说明如何应用
- [核心优势 2] - 具体说明如何应用
- [核心优势 3] - 具体说明如何应用

期待有机会进一步沟通，
详细讨论我如何能为贵公司贡献力量。

此致
敬礼！

[姓名]
[电话]
[邮箱]
[LinkedIn/GitHub/个人网站]
[日期]
```

## 按公司类型分类

### 大厂（字节/阿里/腾讯等）
```
重点：
- 技术深度和广度
- 大规模系统经验
- 高并发/高性能经验
- 团队协作和跨部门沟通

语气：专业、自信、不卑不亢
```

### 创业公司
```
重点：
- 多面手能力（全栈/多技能）
- 从 0 到 1 经验
- 快速学习和适应
- 创业心态（ownership）

语气：热情、灵活、愿意承担
```

### 外企
```
重点：
- 英语能力（如有）
- 国际化视野
- 工作流程规范
- Work-Life Balance 理念

语气：专业、直接、结果导向
```

### 国企/事业单位
```
重点：
- 稳定性
- 责任心
- 团队合作
- 长期发展意愿

语气：稳重、诚恳、务实
```

## 按投递渠道分类

### 官网投递
```
开头：
"我在贵公司官网看到了 XX 职位的招聘信息..."

特点：正式、规范、完整
```

### 内推
```
开头：
"经 [推荐人姓名] 推荐，我了解到贵公司 XX 职位..."

特点：提及推荐人、增加信任度
```

### 猎头
```
开头：
"通过 [猎头公司/猎头姓名]，我了解到贵公司 XX 职位..."

特点：专业、高效、突出匹配度
```

### LinkedIn/脉脉
```
开头：
"我在 LinkedIn 上关注贵公司已久，看到 XX 职位开放..."

特点：展示对公司的长期关注
```

## 常用金句

### 开场白
- "贵公司一直是行业标杆，我非常期待能加入..."
- "作为贵公司产品的忠实用户，我很兴奋看到..."
- "经朋友推荐，我了解到贵公司正在..."

### 表达匹配
- "我的技能与岗位要求高度契合..."
- "过去 X 年的经验让我能够胜任..."
- "我注意到岗位需要 XX 能力，这正是我的专长..."

### 展示成就
- "在 XX 项目中，我主导了..."
- "通过优化 XX，我实现了..."
- "我带领团队完成了..."

### 结尾
- "期待有机会详细讨论我的贡献"
- "感谢您的时间和考虑"
- "希望能有机会加入贵公司"

## 避坑指南

### ❌ 不要写
- "我相信我能胜任这个职位"（空洞）
- "我是一个完美主义者"（陈词滥调）
- "我希望贵公司能提供..."（只谈索取）
- "我急需一份工作"（ desperation）
- "我的前公司很糟糕"（负面评价）

### ✅ 要写
- "我在 XX 项目中实现了 XX 成果"（具体）
- "我通过 XX 方法解决了 XX 问题"（案例）
- "我能为团队带来 XX 价值"（贡献）
- "我一直关注贵公司在 XX 领域的发展"（了解）
- "我的 XX 经验与岗位要求匹配"（匹配）

## 格式规范

### 邮件主题
```
[职位名称] 申请 - [姓名] - [核心优势]
例：高级前端工程师申请 - 张三 - 5 年 React 经验
```

### 邮件正文
```
- 称呼：尊敬的招聘团队/HR/具体姓名
- 正文：简洁版求职信（200-300 字）
- 附件：简历（PDF 格式）
- 签名：联系方式 + LinkedIn/GitHub
```

### 在线表单
```
- 按字段要求填写
- 不要复制粘贴大段文字
- 利用"其他信息"字段补充亮点
```

---

**使用说明：**
1. 根据目标公司选择合适的模板
2. 填充具体信息和案例
3. 用 humanizer 润色，去除模板感
4. 针对每家公司定制，不要一封通用
5. 检查公司名称和职位名称不要写错

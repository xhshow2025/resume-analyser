# 自我介绍模板库

---

## 通用模板

### 30 秒版（电梯演讲）
```
大家好，我是 [姓名]，[工作年限] 年 [核心领域] 经验。
目前在 [当前公司] 负责 [核心工作]，
主导过 [最亮眼项目]，实现 [量化成果]。
我擅长 [核心技能 1]、[核心技能 2]、[核心技能 3]，
正在寻找 [目标职位] 岗位，
希望能用我的 [核心能力] 帮助贵公司 [创造价值]。
```

### 1 分钟版（面试开场）
```
面试官您好，我是 [姓名]，毕业于 [学校][专业]。

过去 [工作年限] 年我一直专注于 [领域]，
目前在 [公司] 担任 [职位]，负责 [工作内容]。

我的核心优势有三点：
第一，[优势 1 - 技术/经验]。[具体例证]。
第二，[优势 2 - 成就/项目]。[量化成果]。
第三，[优势 3 - 软技能/特质]。[具体说明]。

我看过贵公司的岗位描述，与我的技能和经验非常匹配。
希望能有机会加入，一起 [共同目标]。
```

### 3 分钟版（详细陈述）
```
面试官您好，我是 [姓名]，非常感谢给我这次面试机会。

【教育背景】
我毕业于 [学校][专业]，[学历]。
[在校期间的亮点，如 GPA、奖项、项目等]。

【工作经历】
[按时间倒序介绍每段工作经历]

第一份工作是在 [公司 1]，[职位 1]。
这段经历让我 [收获/成长]。

目前在 [公司 2] 担任 [职位 2]，
主要负责三块工作：
1. [工作内容 1]
2. [工作内容 2]
3. [工作内容 3]

【项目成就】
最想分享的是 [项目名称]。
当时面临 [挑战/背景]，
我通过 [行动/方案]，
最终实现 [成果 - 量化数据]。
这个经历让我 [收获/成长]。

【技术专长】
我的核心技术栈是 [技术栈]。
除了技术深度，我也注重广度：
- [相关技能 1]
- [相关技能 2]
- [相关技能 3]

【职业规划】
短期目标（1-2 年）：[具体目标]。
长期目标（3-5 年）[具体目标]。

【结尾】
我看过贵公司的 [产品/技术/文化]，
非常认同 [理念/方向]。
我相信我的 [核心优势] 能够为团队带来价值，
也期待在贵公司实现个人成长。

以上就是我的介绍，谢谢！
```

## 按场景分类

### 技术面试
```
重点突出：
- 技术深度（源码理解、原理掌握）
- 项目难点（技术挑战、解决方案）
- 代码质量（最佳实践、代码审查）
- 技术影响力（开源贡献、技术分享）
```

### HR 面试
```
重点突出：
- 职业规划（与公司发展契合）
- 团队协作（沟通能力、冲突解决）
- 学习能力（自我驱动、快速适应）
- 文化匹配（价值观、工作风格）
```

### 高管面试
```
重点突出：
- 业务理解（行业洞察、商业模式）
- 战略思维（长期规划、资源分配）
- 领导力（团队建设、人才培养）
- 商业价值（ROI、成本意识）
```

## 按职级分类

### 初级工程师（0-3 年）
```
重点：
- 学习能力和成长速度
- 基础技能扎实程度
- 项目参与度和贡献
- 对技术的热情
```

### 高级工程师（3-5 年）
```
重点：
- 独立负责项目的能力
- 技术深度和广度
- 代码质量和最佳实践
- 团队协作和知识分享
```

### 技术专家/架构师（5-8 年）
```
重点：
- 系统架构设计能力
- 技术选型和决策
- 技术难题攻克
- 技术规划和演进
```

### 技术负责人/总监（8 年+）
```
重点：
- 团队建设和管理
- 技术战略和业务结合
- 跨部门协作
- 人才培养和梯队建设
```

## 常用金句

### 开场白
- "我是 XXX，一个对 [技术/领域] 充满热情的 [职位]"
- "过去 X 年，我只专注做一件事：[核心领域]"
- "如果用三个词形容我，那就是：[特质 1]、[特质 2]、[特质 3]"

### 介绍成就
- "我最自豪的项目是..."
- "这个经历让我深刻理解了..."
- "通过这个项目，我学会了..."

### 表达意愿
- "我一直关注贵公司在 [领域] 的发展..."
- "贵公司 [产品/文化] 让我印象深刻..."
- "我相信我的 [能力] 能够为团队带来..."

### 结尾
- "期待有机会深入交流"
- "希望能用我的经验帮助团队"
- "感谢您的时间，期待下一步沟通"

---

**使用说明：**
1. 根据简历内容填充具体信息
2. 用 humanizer 润色，去除模板感
3. 针对面试公司调整重点
4. 练习到自然流畅，不要背诵感
